# 🧠 Smart Text Analyzer (Offline AI App)

An Android application that performs offline text analysis including:

- 😊 Sentiment Analysis (Happy / Sad / Angry / Anxious)
- 📚 Topic Detection (Education, Sports, Technology, etc.)
- 🔑 Keyword Extraction
- ✂️ Auto Text Summarization
- 📊 Visual Sentiment Pie Chart
- 💾 Local History Storage (SQLite)
- 🌙 Dark / Light Theme Support

---

## 🚀 Features

- 100% Offline (No API used)
- Rule-based AI logic using keyword detection
- Fast lightweight processing
- Beautiful Material UI design
- History tracking of all analyzed texts

---

## 🧠 How it works

The app uses a custom keyword-based algorithm:

- Predefined word lists for emotions
- Simple text matching
- Weighted scoring system
- Pie chart visualization using MPAndroidChart

---

## 📊 Libraries Used

- MPAndroidChart (Pie Chart visualization)
- AndroidX Libraries
- SQLite (Local Database)

## 📁 Project Structure

- MainActivity → Core analysis UI
- SentimentAnalyzer → Emotion detection logic
- TopicAnalyzer → Topic classification
- KeywordExtractor → Keyword extraction
- SummaryGenerator → Text summarization
- DatabaseHelper → Local storage (SQLite)
- HistoryActivity → Saved results screen
